== Description ==

Bootstrap is the name of the most familiar front-end framework. Having the most used Bootstrap widgets in elementor is very useful.
"best bootstrap widgets for elementor" plugin has the most used bootstrap widgets as a widget for elementor

Plugin widgets for elementor:
Accordion
Button
badge
Warning
Drop down
Card
Modal

*The main feature of this plugin is its lightness and having the best Bootstrap widgets*
== Installation ==
1. Upload the plugin file to the `/wp-content/plugins/` folder or install it automatically from the WordPress counter
2. Activate the plugin from the plugins section in the WordPress counter
3. Activate elementor
